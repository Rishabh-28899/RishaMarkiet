# RishaMarket
 Short Description / Purpose

This dashboard provides a complete overview of business sales performance by analyzing total revenue, monthly growth, transactions, and returns.It enables stakeholders to monitor KPIs, identify top-performing customers, track monthly trends, and compare performance against targets using interactive visualizations.

Designed for:

Business Analysts

Sales Managers

Finance Teams

Data Analytics learners

3️⃣ Tech Stack

The dashboard was built using:

• 📊 Power BI Desktop – Data visualization and dashboard creation
• 🔄 Power Query – Data cleaning, transformation, and shaping
• 🧠 DAX (Data Analysis Expressions) – Custom measures and time intelligence logic
• 🗂 Data Modeling – Star schema with relationships between:

4️⃣ Data Source

The dataset consists of structured sales and returns data including:

Order transactions

Product details (price, category) etc



5️⃣ Features / Highlights
🔎 Business Problem

Businesses often struggle to:

Track monthly revenue growth

Identify top-performing customers

Monitor return rates

Compare performance against targets

Analyze trends across time

Raw data alone does not provide actionable insights.

🎯 Goal of the Dashboard

To build an interactive and insight-driven Power BI report that:

Tracks Total Revenue and Total Returns

Compares current vs previous month performance

Highlights top customers based on revenue

Monitors transactions and quantity sold

Displays revenue vs target using gauge charts

Enables dynamic filtering using slicers and Top N filters

📊 Walkthrough of Key Visuals
🔹 KPI Cards

Total Revenue

Total Transactions

Total Returns

Quantity Sold

Previous Month Revenue

🔹 Monthly Revenue Trend (Line/Column Chart)

Displays revenue performance over time using a Calendar table and DAX time intelligence functions like:

PREVIOUSMONTH

DATEADD

CALCULATE

🔹 Gauge Chart

Shows:

Total Revenue

Revenue Target

Latest Month Filter (Top N logic applied)

🔹 Top Customer Analysis

Ranks customers based on revenue using DAX measures and filtering logic.

🔹 Product & Category Breakdown

Revenue and quantity analyzed by:

Product Category

Price Tier (High / Mid / Low classification)

🔹 Returns Analysis

Tracks total returns and compares against sales to identify operational issues.

💡 Business Impact & Insights

📈 Performance Monitoring
Management can track revenue growth and monthly trends instantly.

🎯 Target Tracking
Gauge visual helps evaluate whether business goals are being achieved.

👥 Customer Insights
Identify high-value customers contributing maximum revenue.

📦 Product Optimization
Understand which product tiers generate the most revenue.

🔁 Operational Efficiency
Monitor return rates to improve product quality and logistics.
